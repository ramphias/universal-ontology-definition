# 奢侈品行业 Addon | Luxury Goods Industry Addon

## 概述

奢侈品行业 Addon 继承 L1 通用 Ontology，新增了奢侈品行业特有的概念体系，覆盖：

- 👜 **品牌与产品** — 奢侈品牌、品牌屋、商品、系列、SKU 变体
- 🏪 **渠道** — 精品店、百货专柜
- 💎 **客户体验** — 客户顾问、VIP 客户、一对一私享服务
- 🔗 **供应链与溯源** — 原料批次、工艺证书、溯源事件
- 🛡️ **真伪与风控** — 真伪证书、假货风险、数字护照
- 📏 **运营规则** — 稀缺性分配规则、售后养护
- 🎯 **核心能力** — 客户经营能力、数字护照能力、精品配货能力
