# 咨询行业 Addon | Consulting Industry Addon

## 概述

咨询行业 Addon 继承 L1 通用 Ontology，新增了咨询行业特有的概念体系，覆盖：

- 🏢 **组织结构** — 咨询公司、业务线、能力中心
- 📋 **项目管理** — 签约项目、工作流、交付物
- 🧠 **方法论与框架** — 方法论、分析框架
- 👥 **人才体系** — 合伙人、总监、项目经理、顾问、分析师、领域专家
- 🤝 **客户关系** — 客户账户、客户关系、商机管道
- 📊 **运营指标** — 利用率、计费费率
- 📚 **知识管理** — 知识资产、案例研究、思想领导力
- ⚖️ **治理与风控** — 独立性政策、保密政策、利益冲突风险、范围蔓延风险
